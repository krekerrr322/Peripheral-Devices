## 1. Загальна характеристика SSD

**SSD (Solid State Drive)** — це твердотільний накопичувач даних, який використовує мікросхеми NAND Flash пам’яті для зберігання інформації.

На відміну від HDD:
- не має механічних рухомих частин
- працює безшумно
- має значно менший час доступу до даних
- споживає менше електроенергії
- стійкіший до вібрацій і ударів

### Основні переваги SSD:
- Висока швидкість читання та запису
- Низька латентність (0.01–0.1 мс)
- Менший нагрів
- Компактність

### Основні недоліки:
- Обмежена кількість циклів перезапису
- Вища вартість за 1 ГБ порівняно з HDD

# 2. Типи SSD за інтерфейсом підключення

---

## 2.1 SATA SSD (2.5")

Інтерфейс: SATA III (6 Гбіт/с)  
Форм-фактор: 2.5 дюйма  

Максимальна швидкість:
- Читання: ~550 МБ/с
- Запис: ~520 МБ/с

Обмеження швидкості пов'язане саме з пропускною здатністю SATA.

### Приклад:
Samsung 870 EVO

![SATA SSD](images/sata_ssd.jpg)

---

## 2.2 M.2 SATA SSD

Форм-фактор: M.2  
Інтерфейс: SATA  

Має компактний вигляд (плата без корпусу).

Швидкість така ж як у SATA 2.5", оскільки використовує той самий протокол.

### Розміри:
- 2242
- 2260
- 2280 (найпоширеніший)
- 22110

![M.2 SATA SSD](images/m2_sata.jpg)

---

## 2.3 NVMe SSD (PCIe)

Інтерфейс: PCI Express  
Протокол: NVMe (Non-Volatile Memory Express)

Переваги NVMe:
- Паралельна обробка запитів
- Низька затримка
- Висока пропускна здатність

### Покоління PCIe:

| Версія PCIe | Швидкість |
|------------|------------|
| PCIe 3.0 x4 | до 3500 МБ/с |
| PCIe 4.0 x4 | до 7000 МБ/с |
| PCIe 5.0 x4 | до 14000 МБ/с |

### Приклад:
Samsung 980 PRO

![NVMe SSD](images/nvme_ssd.jpg)

# 3. Типи NAND пам’яті

---

## 3.1 SLC (Single-Level Cell)

- 1 біт на комірку
- Найвища швидкість
- Найбільший ресурс перезапису
- Дуже дорога

---

## 3.2 MLC (Multi-Level Cell)

- 2 біти на комірку
- Баланс між ціною та довговічністю

---

## 3.3 TLC (Triple-Level Cell)

- 3 біти на комірку
- Найпоширеніший тип
- Хороше співвідношення ціна/об’єм

---

## 3.4 QLC (Quad-Level Cell)

- 4 біти на комірку
- Дешевша
- Менший ресурс перезапису

---

# 4. Основні технічні характеристики SSD

---

## 4.1 Об’єм (Capacity)

- 120 ГБ
- 240 ГБ
- 500 ГБ
- 1 ТБ
- 2 ТБ
- 4 ТБ+

---

## 4.2 Швидкість читання та запису

Вимірюється в МБ/с.

Розрізняють:
- Послідовне читання
- Послідовний запис
- Випадкове читання (IOPS)
- Випадковий запис (IOPS)

---

## 4.3 TBW (Total Bytes Written)

Показує, скільки даних можна записати на SSD за весь період експлуатації.

Наприклад:
- 150 TBW
- 300 TBW
- 600 TBW

---

## 4.4 MTBF (Mean Time Between Failures)

Середній час напрацювання на відмову.

Зазвичай:
- 1 500 000 – 2 000 000 годин

---

## 4.5 Наявність DRAM-кешу

DRAM-кеш:
- Підвищує продуктивність
- Зменшує зношування NAND

DRAM-less моделі дешевші, але повільніші.

## 5. Скрипт для відображення інформації про SSD

![Плата захоплення](https://i.postimg.cc/t4t9m7TH/Znimok-ekrana-z-2026-02-20-12-31-46.png)

import subprocess
import platform
import json


def get_windows_disks():
    try:
        # PowerShell команда для отримання інформації про фізичні диски
        command = [
            "powershell",
            "-Command",
            "Get-PhysicalDisk | Select FriendlyName, MediaType, Size, HealthStatus | ConvertTo-Json"
        ]

        result = subprocess.run(command, capture_output=True, text=True)

        if result.returncode != 0:
            print("Помилка виконання PowerShell.")
            print(result.stderr)
            return

        disks = json.loads(result.stdout)

        if not isinstance(disks, list):
            disks = [disks]

        print("\n=== Інформація про накопичувачі ===\n")

        for disk in disks:
            print(f"Назва: {disk.get('FriendlyName')}")
            print(f"Тип носія: {disk.get('MediaType')}")
            print(f"Обʼєм (GB): {round(int(disk.get('Size')) / (1024**3), 2)}")
            print(f"Стан: {disk.get('HealthStatus')}")
            print("-" * 40)

    except Exception as e:
        print("Помилка:", e)


def get_linux_disks():
    try:
        command = ["lsblk", "-J", "-o", "NAME,SIZE,TYPE,MODEL"]
        result = subprocess.run(command, capture_output=True, text=True)

        if result.returncode != 0:
            print("Помилка виконання lsblk.")
            print(result.stderr)
            return

        data = json.loads(result.stdout)

        print("\n=== Інформація про накопичувачі ===\n")

        for device in data["blockdevices"]:
            if device["type"] == "disk":
                print(f"Назва: {device.get('name')}")
                print(f"Модель: {device.get('model')}")
                print(f"Розмір: {device.get('size')}")
                print("-" * 40)

    except Exception as e:
        print("Помилка:", e)


def main():
    system = platform.system()

    print(f"Операційна система: {system}")

    if system == "Windows":
        get_windows_disks()
    elif system == "Linux":
        get_linux_disks()
    else:
        print("Непідтримувана ОС.")


if __name__ == "__main__":
    main()
